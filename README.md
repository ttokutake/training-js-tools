# try-lint
=============

## What is *try-lint*?

To try ESLint and Flow.

## Usage

1. Clone this repository.
2. Prepare the environment. You are recommended to use [asdf](https://github.com/asdf-vm/asdf) for installing [node.js](https://nodejs.org/en/).
```bash
$ cd try
$ npm install
$ npm run try src/index.js
$ npm run lint src/*.js
```
